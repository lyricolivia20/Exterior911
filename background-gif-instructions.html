Background GIF Instructions for Exterior 911

This document provides instructions for creating and implementing a background GIF for the Exterior 911 website to achieve a more masculine, rugged aesthetic.

Background GIF Specifications

The website is designed to use a background GIF file named background.gif placed in the /assets/ directory. The CSS is already configured to use this file.

Option 1: Creating a Dynamic Grass Texture

Create a subtle animated grass texture with slight movement, as if a gentle breeze is blowing across a well-maintained lawn.

  • Use dark green tones (forest green, hunter green)
  • Create slight movement animation (3-5 frames)
  • Keep the file size small (under 1MB if possible)
  • Resolution: at least 1920x1080px
  • Subtle texture with minimal distraction

Option 2: Professional Equipment in Action

Create or source a GIF showing professional lawn equipment in action, like a slow-motion close-up of a mower blade cutting grass.

  • Desaturated/muted colors for subtlety
  • Set to low opacity in the CSS (already configured)
  • Slow-motion effect for a premium feel
  • Focus on quality equipment (not cheap consumer models)

Option 3: Texture Pattern Animation

Create a subtle animated pattern using lawn/landscaping textures that slowly transitions or rotates.

  • Combine elements like soil, grass, stone
  • Use angular, geometric transitions
  • Keep movement minimal and professional
  • Dark, rich earth tones

Implementation Instructions

  1. Create or source a GIF file matching one of the options above
  2. Name the file background.gif
  3. Place it in the /assets/ directory
  4. The website CSS is already configured with the proper path and opacity settings
  5. Test the site to ensure the GIF doesn't overpower the content or cause performance issues

Important Notes

  • Keep the file size as small as possible while maintaining quality
  • Ensure the GIF loops seamlessly without visible jumps
  • The animation should be subtle and not distract from the content
  • The GIF should enhance the masculine, professional aesthetic
  • If the GIF causes performance issues, consider converting it to a short looping video instead
